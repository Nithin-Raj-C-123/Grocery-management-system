Nithin Grocery Management System is a web-based grocery billing and inventory management application developed to simplify daily grocery shop operations.
The system allows users to add grocery items with quantity, unit, price, and minimum stock threshold, automatically calculate totals, apply discounts and GST, generate printable bills, and maintain daily sales history.

It also includes advanced features such as low-stock warnings, dark mode, local storage-based data persistence, and daily sales tracking, making it suitable for small retail shops and learning purposes.
